# python_work_demo
# **************************Music Data analysis project**************************
## Objectives
This project analyzes Spotify sing data to identify patterns in:
- Major vs Minor songs.
- Popularity trends.
- Release year distribution.

## Technologies Used
- Python.
- Pandas.
- Numpy.
- Matplotlib.

## Features
- Data cleaning and preprocessing.
- Classification of songs (major/minor)
- Visualization of trends.
- Analysis of artist popularity.
